June Oven Pairing

This plugin pairs directly with your oven over June's cloud — you don't need the June phone app. Click below to start; instructions appear once pairing begins.

  1. On the oven's touchscreen, swipe left twice from the home screen and select Connect. If the oven already shows a list of connected devices instead of a code, tap + to add a new device.
  2. Type the code below into the oven's screen.
  3. Wait for this page to report Paired, then restart Homebridge.
